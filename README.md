# DappNotes

Image file digital time capsule built as a React web app using a Solidity Ethereum blockchain smart contract and Node.js backend


## Usage - Development 

```bash
npm run dev
```

## Usage - Production 

```bash
npm run build
npm run start
```

## Environment
A .env file must be placed in the project root directory with the following variables 

```bash
MONGO_URI=*****************
GOOGLE_ANALYTICS_ID=*******
PINATA_API_KEY=************
PINATA_API_SECRET=*********
ALCHEMY_RINKEBY_KEY=*******
ALCHEMY_RINKEBY_URL=*******
RINKEBY_PRIVATE_KEY=*******
```
